# 🚀 Guia Rápido - SIACP

## Como começar em 5 minutos

### Passo 1: Escolha o Prompt
Os prompts estão na pasta `prompts/`. Cada prompt é para um tipo de análise:

- **01-validacao-basica.md** → Validações iniciais da licitação
- **02-checklist-compliance.md** → Checklist completo de conformidade
- **03-analise-conformidade.md** → Análise profunda
- **04-gerar-relatorio.md** → Gerar relatório final

### Passo 2: Copie o Prompt
1. Abra o arquivo do prompt escolhido
2. Clique em **"Raw"** (canto superior direito)
3. `Ctrl + A` → `Ctrl + C` (copiar tudo)

### Passo 3: Cole no ChatGPT
1. Acesse https://chatgpt.com
2. Cole o prompt (`Ctrl + V`)
3. Adicione seus dados da licitação
4. Envie

### Passo 4: Use o Template
- O ChatGPT vai responder dentro do template estruturado
- **NÃO altere o formato**
- Se precisar melhorar, use o mesmo template

### Passo 5: Salve o Resultado
- Copie a resposta
- Crie um arquivo no seu repositório com o resultado
- Ou salve em um arquivo local
